# Customer Retention & Churn Analytics Dashboard

## 📌 Project Overview
This project analyzes customer churn behavior and retention trends for a subscription-based business using Power BI.

The objective is to identify:
- Churn patterns
- Customer retention drivers
- Customer lifetime trends
- Business recommendations to reduce churn

---
## 🛠 Tools Used
- Power BI
- Power Query (for basic data cleaning and transformation)

---

## 📊 Key Insights
- Month-to-month contract customers showed highest churn
- Electronic check payment users had higher churn rates
- Long-term customers were more likely to stay
- Two-year contracts had strongest retention

---

## ✅ Recommendations
- Encourage long-term subscription plans
- Improve onboarding experience
- Promote automatic payment methods
- Introduce loyalty rewards programs
